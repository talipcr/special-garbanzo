# Special-garbanzo-node

Special-garbanzo-node is an API RESTful developed in Node/Express.js for demonstration.

## Installation

Use NPM to start

```bash
npm install
npm run dev
```

For lint 
```bash
npm run lint
npm run lint:fix
```

## Requirement

You need to install [MongoDB](https://docs.mongodb.com/manual/installation/)
